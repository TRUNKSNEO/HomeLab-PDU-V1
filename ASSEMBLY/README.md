## Sub-Assemblies:
Some variants have sub-assemblies that are shared.


<br>

### PCB Tray:
All non-3D printed variants use the PCB tray sub-assembly. This tray makes soldering the daughter boards easier, and allows the board to be removed from the chassis easier.

### IEC Plug:
Due to the extra depth allowed on full racks, all 19" variants use the IEC plug sub-assembly to allow for a detachable cable.
