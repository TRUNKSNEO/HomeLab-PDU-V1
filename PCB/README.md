# PCB:
All variants use the same 2 layer 1.6mm thick PCB. This folder contains production files that can be uploaded to a PCB service for manufacturing, JLCPCB is my go-to. PCB assembly is recommended unless you have capability to do reflow, or hot air soldering.
There is currently no option for a through-hole variant because the tranistor does not come in a TH package.
